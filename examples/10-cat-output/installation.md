# Installation

## Using npm

```bash
npm install my-project
```

## Using yarn

```bash
yarn add my-project
```

## From Source

```bash
git clone https://github.com/example/my-project.git
cd my-project
npm install
npm run build
```

## Verify Installation

```bash
my-project --version
```

## Troubleshooting

If you encounter issues, check the documentation or file an issue on GitHub.
